# Smart Expense Tracker

## 📌 Overview
A Python-based Smart Expense Tracker with GUI and CSV storage.  
It helps users record daily expenses, categorize them, and view reports.

---

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/your-username/Smart-Expense-tracker.git
cd smart-Expense-Tracker
2. Install Dependencies
Make sure Python 3 is installed.
Install required libraries:
pip install pandas tkinter

3. Run the Application
python main.py

📂 How It Works
- Expenses are stored in expenses.csv.
- GUI allows users to:
- Add new expense entries (date, category, amount, notes).
- View all expenses in a table.
- Generate simple reports (tota


🚀 Features
- Add, edit, and delete expenses.
- Save data in CSV format.
- Simple GUI for easy interaction.
- Category-wise expense tracking.
